# Archivo base / Objetivo de diseño

Este documento es la **referencia objetivo** hacia la que iteramos en el loop.
La imagen base la aportó el usuario: landing page de **"Cobalt"** (fintech, estilo dark/espacial).
Screenshot objetivo guardado en: `screenshots/referencia-cobalt.png` (recreación descriptiva abajo).

## Estilo visual de Cobalt (a replicar)

- **Fondo:** azul-negro muy oscuro (#05060a → #0a0e17) con glows radiales azules en la parte superior y transiciones suaves. Fondo tipo "campo de estrellas" sutil.
- **Tipografía:** sans-serif bold, tracking ajustado. Hero enorme (~64px), dos líneas.
- **Texto:** títulos blancos, cuerpo gris apagado (#8b95a5).
- **Acentos:** azul/cian (#4a9eff), verde para valores positivos (badges), línea de gráfico cian/teal.
- **Tarjetas:** fondo oscuro con borde sutil (rgba(255,255,255,0.06)) y ligero gradiente.
- **Botones:** "Join the waitlist" = pill claro/blanco con texto oscuro; secundario = texto con flecha.

## Estructura (secciones de arriba a abajo)

1. **Nav:** logo (izq), Blog + botón "Join the waitlist" (der). Minimal.
2. **Hero:** centrado. Título grande a 2 líneas, subtítulo, 2 CTAs (primario pill + "Learn more ↓"). Glow azul arriba.
3. **Mockup de dashboard:** tarjeta grande redondeada con sidebar (Insights, Company, Transactions, Cards, Accounting) + stat cards + gráfico de líneas Income/Expenses.
4. **"Who said finance has to be boring?"** — 2 columnas (título izq, párrafo der).
5. **"Everything you need. Nothing you don't"** — título + grid de feature cards (3 arriba, 2 abajo con logos de integraciones).
6. **"Meet Genius"** — asistente IA. 2 tarjetas: Smart forecasting (gráfico + alerta), Chat con IA.
7. **CTA final:** título grande + subtítulo + botón waitlist.
8. **Footer:** logo, links, iconos sociales, texto legal.
